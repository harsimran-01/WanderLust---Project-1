## рџ“Њ Overview

**WanderLust** is a full-stack web application where users can explore travel stays and property listings in a clean and interactive interface.

It is inspired by platforms like **Airbnb** and was built to practice **full-stack web development concepts**, including:

- Server-side rendering
- CRUD operations
- Routing
- Middleware
- Database integration
- Error handling
- Cloud image configuration

This project helped me understand how a complete web application is structured and managed from **frontend views to backend logic**.

## рџ“‚ Folder Explanation

### `controllers/`
Contains the main logic for handling routes and application functionality.

### `models/`
Includes MongoDB/Mongoose models used to define data structure.

### `routes/`
Contains all route definitions for listings and application navigation.

### `views/`
EJS templates used to render frontend pages dynamically.

### `public/`
Stores static assets such as CSS, JS, and images.

### `utils/`
Contains utility/helper functions used across the project.

### `init/`
Used for initializing or seeding the database with sample data.

### `middleware.js`
Custom middleware functions for validation and error handling.

### `cloudConfig.js`
Handles cloud-based configuration for images or uploads.

### `schema.js`
Validation schema for data handling and form validation.
